nodepod - shared FS attach from sibling worker

Demo of Nodepod.attachFS(sab). Main thread boots nodepod, creates a plain sibling Web Worker (not one nodepod spawned via spawn()), and postMessages it the shared VFS buffer. The sibling reads files straight out of the same SharedArrayBuffer, no IPC per call. When main writes a new file the sibling sees it on the next read because they're looking at the same memory. [gold = main, purple = sibling]
