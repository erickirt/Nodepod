nodepod - tailwind v3 + vite 8 + react reproduction

Mirrors a real-world ai-landing template: tailwindcss@^3.4.7 + postcss@^8.4.39 + autoprefixer@^10.4.19 + vite@^8.0.8 + react@^18.3.1. Tailwind v3 is pure JS (no WASM), so any 5xx on /src/index.css isolates a nodepod-side bug rather than a Tailwind-Oxide WASM issue.

The "v8 + jsx-runtime" button uses "jsx": "react-jsx" (no import React needed). The "v7 baseline" button uses vite@^7.3.1 as a sanity check that the same project works on the older bundler.
